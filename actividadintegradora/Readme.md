# Mi Página Web Personal

Este proyecto consiste en el desarrollo de una página web básica utilizando HTML y CSS, como parte del aprendizaje de programación web.

## Desarrollo del proyecto

Para la realización de esta página, comencé creando la estructura base en HTML, utilizando etiquetas semánticas como header, main, section y footer para organizar el contenido.

Luego agregué las diferentes secciones solicitadas:

- Sobre mí: donde describo quién soy y qué estudio
- Intereses: donde incluyo una lista de mis gustos como videojuegos, música y deportes
- Formación: donde menciono lo que estoy aprendiendo actualmente y lo que me gustaría aprender en el futuro
- Contacto: donde implementé un formulario con campos de nombre, correo y mensaje

También implementé un menú de navegación con enlaces internos para facilitar el desplazamiento dentro de la página.

Posteriormente, añadí estilos básicos con CSS, aplicando color de fondo, color de texto y márgenes para mejorar la presentación visual.

## Tecnologías utilizadas

- HTML
- CSS
